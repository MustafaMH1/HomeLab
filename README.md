# 🏡 Home Lab Project

This repository contains the documentation and configuration for my **Home Lab project**.  
It covers everything from **hardware setup, Linux installations, virtualization (Proxmox, Docker, Kubernetes), networking, storage, automation (Ansible), monitoring, and security**.  

The goal of this project is to:  
- Learn and experiment with enterprise technologies in a personal lab.  
- Maintain structured documentation for repeatable setups.  
- Build a foundation for automation and Infrastructure as Code (IaC).
